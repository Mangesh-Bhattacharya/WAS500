Author: Jon Duckett

ISBN: 978-111854184-3

Pages: 552

Published: 2014

Language: English

Genre: Web Development

Details


Every day, more and more people want to learn some HTML and CSS. Joining the professional web designers and programmers are new audiences who need to know a little bit of code at work and those who want to make their personal blogs more attractive.

Many books teaching HTML and CSS are dry and only written for those who want to become programmers, which is why this book takes an entirely new approach.
